# Documentation

This `docs/` folder contains the project's documentation and guides. If you're contributing, edit the files here and open a PR.

## Contents

- `QUICKSTART.md` — Quick start and setup for users and developers
- `QUICK_REFERENCE.md` — Short developer reference and common tasks
- `API_QUICK_REFERENCE.md` — API usage examples and data models
- `TESTING_GUIDE.md` — Testing scenarios and checklists
- `STRUCTURE.md` — Project structure and data flow
- `STRUCTURE_VISUAL_GUIDE.md` — Visual diagrams of the data model and flows
- `UPDATE_NOTES.md` — Release notes and notable changes
- `DEPLOYMENT.md` — Build and deployment instructions for Windows
- `FRONTEND_IMPLEMENTATION.md` — Frontend implementation details and examples
- `BUILD.md` — Build steps and troubleshooting tips
- `CSV_BULK_UPLOAD_GUIDE.md` — CSV import/export instructions and samples
- `DATABASE_CHANGES_SUMMARY.md` — Summary of schema changes and migration notes
- `DECIMAL_QUANTITY_MIGRATION.md` — Notes for decimal quantity migration
- `IMPLEMENTATION_CHECKLIST.md` — Implementation and testing checklist
